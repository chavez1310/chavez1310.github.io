  • Arrays and Hashes

    Arrays and Hashes are, from what I can tell so far, similar, but still very different aspects of Ruby. For one thing they are both indexed collections, and both collections of objects can be accessed using a key. However, with arrays, the key you use to access the value in the array is an integer, while with hashes you can assign keys yourself, and call the value by calling the key.
